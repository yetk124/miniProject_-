📊 서울 범죄 & 부동산 데이터 분석

서울시의 부동산 실거래가, CCTV 개수, 범죄 발생 데이터를 분석하고 시각화합니다.

📌 프로젝트 개요

  • 🏠 부동산 가격과 범죄율 관계 분석
  • 📷 CCTV 개수와 치안 비교
  • 🤖 머신러닝을 활용한 범죄 예측 모델
  • 🌍 웹 서비스 시각적 분석 제공

💾 사용 데이터

데이터셋설명
🏠 부동산 실거래가서울시 지역별 평균 거래 금액
📷 CCTV 데이터서울 자치구별 CCTV 개수
🚔 범죄 데이터5대 범죄(살인, 강도, 절도, 폭력, 강간) 통계
🌍 위경도 데이터지도 시각화를 위한 위치 정보
🚨 치안시설 데이터서울 자치구별 치안시설 개수

⚙️ 사용한 기술

  • 📊 데이터 분석 & 시각화: Pandas, Matplotlib, Seaborn, Folium, Plotly
  • 🧠 머신러닝: Scikit-learn (랜덤 포레스트, 회귀 분석), XGBoost
  • 🌐 웹 개발: Flask, HTML, CSS, JavaScript
  • 🗄 데이터베이스: MongoDB (데이터 저장 및 관리)
